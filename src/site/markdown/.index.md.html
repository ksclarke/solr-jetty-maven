Introduction

Solr-Jetty-Maven is a simple Maven wrapper around the Solr search service. It allows Solr to be run from within a Maven project using Jetty as the servlet container and Web server. It's designed to provide a drop dead simple use of Solr but also includes init.d scripts so that it can be used as a system service on a Linux machine.
